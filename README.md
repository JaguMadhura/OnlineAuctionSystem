# 🔨 Online Auction System (eBay-lite)

A Full-Stack Spring Boot application where users can list items for auction and place live bids.

## 🚀 Features
- **Secure Login:** Session-based login (Demo: admin / admin123).
- **List Items:** Add new products with a starting price.
- **Live Bidding:** Automatically tracks the highest bidder and price.
- **Dynamic UI:** Responsive design using Tailwind CSS.

## 🛠️ Tech Stack
- **Backend:** Java 17, Spring Boot, Spring Data JPA.
- **Frontend:** Thymeleaf, HTML5, Tailwind CSS.
- **Database:** H2 (In-memory database).

## 🏃 How to Run
1. Clone the repository.
2. Open in Eclipse (STS) as a Maven Project.
3. Run as `Spring Boot App`.
4. Open `http://localhost:8080` in your browser.
